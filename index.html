O que eu procuro

Eu, como programador praticante procuro me inserir no mercado de trabalho de desenvolvimento
web back-end inicialmente como júnior, mas ainda sim procuro estudar a parte de front-end, abaixo estão as
tecnologias com as quais eu me interesso em aprofundar e aprender constantemente como
desenvolvedor júnior, caso contratado:

Python: A linguagem Python, apesar
de ser uma das requisitas no mercado
nos dias atuais, é a que mais me chamou a atenção
por sua praticidade, principalmente no que se
refere ao framework Django.

Django: O Django é um framework para desenvolvimento web
com Python, ela usa a arquitetura MVT (Model, View and Templates)
e com isso me interessei por sua simplicidade e capacidade resolver
diversos problemas com a programação back-end.

HTML5 e CSS3: Por serem a base de todo o desenvolvimento web
é de extrema importância para o desenvolvimento com Django e web, afim
de deixar qualquer sistema com uma boa aparência e que atenda
os requisitos pedidos.

Conhecimentos que possuo:
